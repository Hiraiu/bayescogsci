An Introduction to Bayesian Data Analysis for Cognitive Science

9.1 A first attempt at analyzing 0,1 responses

How can we analyze these data? As a first attempt, we can try the linear modeling approach we have just learnt. Assuming a Normal likelihood, we could just fit the following model:

m<-lm(correct~c_set_size,df_recall_data)
summary(m)
## 
## Call:
## lm(formula = correct ~ c_set_size, data = df_recall_data)
## 
## Residuals:
##     Min      1Q  Median      3Q     Max 
## -0.9304 -0.0522  0.0696  0.1913  0.3130 
## 
## Coefficients:
##             Estimate Std. Error t value Pr(>|t|)
## (Intercept)   0.8696     0.0325   26.78  < 2e-16
## c_set_size   -0.0609     0.0145   -4.19  6.5e-05
## 
## Residual standard error: 0.311 on 90 degrees of freedom
## Multiple R-squared:  0.163,  Adjusted R-squared:  0.154 
## F-statistic: 17.6 on 1 and 90 DF,  p-value: 6.46e-05
alpha_hat<-round(summary(m)$coefficients[1,1],2)
beta_hat<-round(summary(m)$coefficients[2,1],2)
sigma_hat<-round(sigma(m),2)

On the surface, this model looks fine: the intercept reflects the grand mean in the data:

## grand mean accuracy:
mean(with(df_recall_data,
     tapply(correct,c_set_size,mean)))
## [1] 0.86957

The estimate of the slope tells us that every unit increase in set size is estimated to reduce the proportion of accurate responses by 0.06. Formally, the model is:

\[\begin{equation} correct \sim Normal(0.87} + -0.06 \times c\_set\_size,0.31) \end{equation}\]

Stated in this way, one problem with this model becomes apparent: a Normal likelihood can produce values on the real number line ranging from \(-\infty\) to \(+\infty\). However, the dependent variable can only have values 0 or 1.

Is there a more sensible model that can be defined? The generalized linear modeling framework addresses this problem with the standard linear model.
